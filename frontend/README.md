# PitchZone Frontend

React + Vite + Tailwind CSS 4.

Tài liệu đầy đủ (cài đặt, API, VNPay, ảnh giao diện): xem [README.md](../README.md) ở thư mục gốc dự án.

## Lệnh nhanh

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run lint
```

Copy `frontend/.env.example` → `.env` trước khi chạy.
